# qbx_garages_radialmenu
radial-menu Inspired Garage Script for QBX

Dependencies
qbx_radialmenu
qbx_core

Installation:
Delete qbx_garages.
Drag the downloaded qbx_garages folder into the [qbx] folder.
And insert the SQL rows into the player_vehicles table with the given one or add manually like this and remember to make the license as a "KEY" in the databse
